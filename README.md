## 📌 Project Overview

The FMCG industry operates in an intensely competitive environment where brands must continuously monitor market dynamics, consumer preferences, and competitor activity.

This project was developed as a self-initiated business analytics dashboard to demonstrate how brand intelligence can be consolidated into a single decision-support interface.

The dashboard provides a structured view of key performance indicators, market trends, brand metrics, and strategic insights that marketing teams can leverage for informed decision-making.

---

## 🎯 Business Objective

The primary objective of this project is to showcase how a centralized dashboard can help organizations:

* Monitor brand performance indicators
* Track market trends and category developments
* Analyze competitive positioning
* Support strategic marketing decisions
* Improve visibility into key business metrics
* Enable data-driven planning and execution
